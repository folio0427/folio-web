# 發帖、報名與配對流程

[Folio](./folio.md) 不用滑卡，改用「發帖 + 報名」：

1. **發帖（Post）**：在探索頁發帖徵共讀夥伴，填書名、為什麼想讀、想找怎樣的人、
   目前進度、預期週數。發帖有每日配額（節奏提醒，非反濫用）。
2. **報名（Apply）**：看到帖子的人報名，鏡像填寫（書名自動帶入、為什麼想加入、
   我能成為怎樣的夥伴、預期週數）。同一帖只能報一次。
3. **答應（Accept）**：發帖者可從報名者中**答應多人**（1:N）；
   每答應一人就[建立一間共讀室](./co-reading-room.md)。略過則靜默淘汰。
4. 帖子到期自動關閉、不再接受報名。

## 與心態的關係

只有[過濾矩陣](./modes.md)允許互看的人，才會在探索頁看到彼此的帖子。
