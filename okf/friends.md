# 好友系統

[Folio](./folio.md) 在配對機制之外提供輕量好友功能：

- **好友申請**：可送出 / 接受好友申請；交友申請以紅點提示（好友 chip）。
- **共讀邀請**：可邀請好友一起共讀某本書 → 直接[開共讀室](./co-reading-room.md)；
  邀請顯示在共讀室列表頂部，誤觸關掉有草稿暫存（不丟字）。
- **分級推播**：好友申請 / 共讀邀請走分級 FCM 推播。
- **共讀室內加好友**：在共讀過程中可把對方加為好友。
