# Folio — Open Knowledge Bundle

> Folio（以書交友 / a book-based friend-matching app）的產品知識包，供 AI agent / LLM 取用。
> 官方網站 https://foliomatch.app ・ bundle id `com.yuliao.folio`
> 本包描述「Folio 這個 app」，與英文 folio（對開本／印刷術語）無關 — 見 [品牌與命名](./brand.md)。

## 產品核心

* [Folio 是什麼](./folio.md) - 以書交友的慢步調社交 app，書是對話入口而非閱讀器。
* [品牌與命名](./brand.md) - 「Folio」(app) 與英文 folio (印刷術語) 的區別、官方資產。
* [產品差異點](./differentiators.md) - 無滑卡、初期無照片、進度閘解鎖、質 > 量。

## 機制

* [心態模式與過濾矩陣](./modes.md) - 純粹書友 / 純粹找緣分 / 不拘。
* [發帖、報名與配對流程](./matching-flow.md) - 發帖徵共讀夥伴 → 報名 → 開共讀室。
* [共讀室與進度閘](./co-reading-room.md) - 聊天起始鎖定，雙方各達 25% 才解鎖。
* [進度模型 User × Book](./progress-model.md) - 一本書一個全局進度，跨房間 + 書房同步。
* [便利貼與讀後分享卡](./notes-and-reflections.md) - 共讀心得紙條 + 100% 解鎖的分享卡。
* [好友系統](./friends.md) - 好友申請、共讀邀請、分級推播。

## 信任與安全

* [安全、隱私與年齡限制](./safety-and-privacy.md) - 18+、檢舉封鎖、隱私保護、訊息防刷。
