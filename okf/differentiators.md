# 產品差異點

[Folio](./folio.md) 刻意與主流交友 app 反向設計：

| 主流交友 app | Folio |
|---|---|
| 滑卡 / 照片優先 | **無滑卡**、初期**無照片** — 看書、看心得、看人 |
| 配對即可聊 | [共讀室起始鎖定](./co-reading-room.md)，雙方各讀到 25% 才解鎖 |
| 衝量、快配對 | **質 > 量**、**儀式感勝過效率**、慢步調 |
| 純交友／純約會二選一 | [三種心態](./modes.md)：純粹書友 / 純粹找緣分 / 不拘 |
| 必須配對才有用 | **個人書房可單獨使用**，配對是 opt-in |

## 為什麼這樣設計

- **進度閘篩選誠意**：願意一起讀到 25% 並[寫下心得](./notes-and-reflections.md)的人，
  才解鎖聊天 — 留下的關係更深。
- **書當話題擔保**：不靠搭訕話術，共讀過程本身就是關係。
- **無照片降低外貌偏誤**：先因為書與想法連結，而非長相。
